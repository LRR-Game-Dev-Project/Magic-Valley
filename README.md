# עמק הפלאות

**Tagline:** “בעמק הפלאות — כל טעות היא קסם חדש.”

## תיאור המשחק
*עמק הפלאות* הוא משחק הרפתקה עלילתית צבעוני וקליל, בו השחקן מגלם דמות רגילה שנקלעת לעמק קסום מלא יצורים מוזרים ותושבים מצחיקים. במהלך המשחק, השחקן:
- חוקר אזורים שונים במפה
- מבצע משימות עיקריות וצדדיות
- מתמודד בקרבות עם אויבים ובוסים
- משדרג יכולות ונשקים תוך כדי התקדמות בעלילה  

המשחק מיועד לנוער ומבוגרים צעירים שאוהבים הרפתקאות עם הומור וסיפור.

## פלטפורמה
- מחשב אישי (Keyboard & Mouse)
- מבט מלמעלה על המפה
- עולם דו־ממדי עם גרפיקה צבעונית וידידותית

## דרישות מערכת (מינימום)
- מערכת הפעלה: Windows 10 / macOS 11  
- מעבד: Intel Core i3 או שווה ערך  
- זיכרון RAM: 4GB  
- כרטיס מסך: Intel HD 4000 או שווה ערך  
- שטח אחסון: 1GB  

## הוראות התקנה
1. הורד את קבצי המשחק מהקישור.  
2. חלץ את הקבצים לתיקייה במחשב שלך.  
3. הפעל את הקובץ `EmekHaPaluot.exe` (Windows) או `EmekHaPaluot.app` (macOS).  
4. עקוב אחרי ההוראות על המסך כדי להתחיל את המשחק.  

## שלבי המשחק
1. **סצנת פתיחה:** הדמות מגיעה לעמק הקסום ומפגשת מדריך שמסביר את חוקי המשחק.  
2. **חקירה והרפתקה:** השחקן חוקר את המפה, משלים משימות צדדיות ומשחק בקרבות מול אויבים.  
3. **שדרוגים:** צובר ניסיון ונשקים חדשים, משדרג את יכולותיו ומתקדם בעלילה.  
4. **סיום:** השחקן מתמודד עם הבוס הסופי ומחזיר את הסדר לעמק הפלאות.  

## מערכות עיקריות
- **מערכת קרב:** פגיעה באויבים נקבעת לפי כוח הנשק ורמת השחקן.  
- **מערכת חקירה:** חקירת האזור מגלה אזורים נסתרים, אוצרות ודמויות חדשות.  
- **מערכת משימות:** משימות עיקריות וצדדיות שמנחות את השחקן ומעשירות את העלילה.  
- **מערכת משאבים:** ניסיון וכלי נשק שמאפשרים שדרוג הדמות.  

## עיצוב הסביבה והדמות
- עולם צבעוני עם טבע, כפרים ועיירות בסגנון ימי הביניים.  
- הדמות הראשית היא ילד חקלאי בתחילת דרכו, לבוש קרוע, שהולך ומתפתח לאורך המשחק.  
- מוזיקה שלווה שמתאימה לאווירת ההרפתקה.
