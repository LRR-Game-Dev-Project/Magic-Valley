# עמק הפלאות
🎯 מה זה?  
משחק הרפתקה דו־ממדי (2D) לשחקן יחיד: אתם מגלמים דמות רגילה שנקלעת לעמק קסום, מלא יצורים מצחיקים ודמויות צבעוניות. השחקן חוקר את המפה, מבצע משימות עיקריות וצדדיות, מתמודד בקרבות מול אויבים ובוסים ומשדרג את יכולותיו תוך כדי התקדמות בעלילה מלאת הומור והרפתקאות.

⚙️ מאפייני משחק בקצרה  
- פלטפורמה: מחשב  
- סגנון גרפי: דו־ממדי (2D) צבעוני ומצויר  
- קהל יעד: נוער ומבוגרים צעירים  
- סוג משחק: שחקן יחיד (Single Player)  

🎮 רכיבים רשמיים  

1) שחקנים  
- מותאם לנוער ומבוגרים צעירים שאוהבים משחקים עם סיפור והומור.  
- חוויה לשחקן יחיד, אינטראקציה אישית עם דמויות המרחב.  

2) יעדים  
- מטרה עיקרית: להביס את הבוס הסופי ולהשיב את הסדר לעמק הפלאות.  
- משימות צדדיות: עזרה לתושבים מוזרים, פתרון חידות קסומות, גילוי אוצרות חבויים.  
- הצגה: סמלים על המפה ודיאלוגים עם דמויות מצביעים על יעד המשימה.  

3) תהליכים (Game Loop)  
**התחלה:** סצנת פתיחה קצרה שמציגה את הגעת הדמות לעמק ומדריך המסביר את חוקי המשחק; HUD עם מד חיים ומפה קטנה.  
**הליבה (Core Loop):**  
חקירה → דיאלוג עם דמויות → משימות/קרבות → קבלת ניסיון וכלי נשק → שדרוג יכולות → חזרה לחקירה.  
**סיום:** הבסת הבוס הסופי; סצנת סיכום עם חזרה של סדר הקסם לעמק.  

4) חוקים  
- חיים מוגבלים: מד חיים מצומצם; אפס חיים = חזרה לנקודת שמירה.  
- נזק לאויבים נקבע לפי כוח נשק ורמת שחקן.  
- השחקן לומד חוקים דרך מדריך וסצנות הדרכה קצרות לאורך המשחק.  

5) משאבים  
- ניסיון: משדרג רמות ופותח יכולות חדשות.  
- כלי נשק: מגוונים ומשפיעים על עוצמת הקרב.  
- השגת משאבים: קרבות, השלמת משימות צדדיות ובוסים.  
- נדירות: מחירים גבוהים לאפשרויות שדרוג; אויבים חזקים.  
- הצגה: HUD מציג מד חיים, ניסיון וכלי נשק פעילים.  

6) עימותים  
- מול המערכת: אויבים קטנים ובוסים.  
- מול שחקנים אחרים: אין.  
- פנימיים: אין.  

7) גבולות העולם  
- סוג עולם: פתוח  
- צורת המפה: שטוחה, דו־ממדית  
- גבולות חזותיים: צוקים והרים טבעיים מסמנים את קצות העמק  
- ניווט: סמלים על המפה, HUD ברור.  

🕹️ שליטה 
- תנועה: חצים במקלדת  
- פעולה: לחיצת מקש לפעולות קרב/אינטראקציה  
- התאמה לפלטפורמה: ניתן למפות כפתורים למקלדת או ג’ויסטיק וירטואלי.
